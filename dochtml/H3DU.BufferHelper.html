H3DU.BufferHelper()

A helper class for accessing and setting data in buffer attributes.

A vertex attribute is made up of values, one for each vertex, each of which is made up of one or more elements, which are numbers such as X coordinates or red components. Each value has the same number of elements.

H3DU.BufferHelper#makeBlank(name, index, count, countPerValue)

Creates an object describing information about a vertex attribute.

Parameters

  • name (Type: number)
    An attribute semantic, such as H3DU.Semantic.POSITION.
  • index (Type: number)
    The set index of the attribute for the given semantic. 0 is the first index of the attribute, 1 is the second, and so on.
  • count (Type: number)
    Number of values. Each value describes the attribute's value for the corresponding vertex.
  • countPerValue (Type: number)
    Number of elements (numbers) for each value.

Return Value

A new vertex attribute with blank values. (Type: Array.<Object>)
